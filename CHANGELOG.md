# Changelog

## v1.0.0 — 2026-04-24

Initial release.

- List, create, edit, and delete short URLs
- Dynamic tag input with existing tag suggestions
- Advanced options: title, valid since/until, max visits, crawlable, forward query
- Visit statistics with bot filtering
- QR code generation (client-side)
- Dashboard widget with recent links and total visit count
- Shlink Web Client integration (servers.csv export for super admins)
- Status badges (expired, max reached, scheduled)
- Full German and English translations
- Statamic permissions: `view short urls`, `edit short urls`
